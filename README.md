## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
```
Make a production build:

```bash
npm run build
# or
yarn build
```

Run production server
```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
